# Diwali Sales Data Analysis

## Project Overview

Performed Exploratory Data Analysis (EDA) on Diwali sales data using Python to understand customer purchasing behavior and identify business opportunities.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings

- Female customers contributed more to sales.
- Customers aged 26–35 generated maximum revenue.
- Uttar Pradesh, Maharashtra, and Karnataka were top-performing states.
- Married women were the most active buyers.
- IT, Healthcare, and Aviation professionals showed strong purchasing behavior.
